# Character Bible Template

Use this to define consistent characters for AI video generation.

## Character: [NAME]

### Physical Description
- Height, build, age
- Hair (color, length, style)
- Face (distinctive features)
- Clothing (signature look)
- Energy/vibe

### Personality
- Core traits
- Speech patterns
- Relationship to other characters

### Prompt Rules
1. Every video prompt must include FULL character description
2. AI has no memory between generations
3. Be specific about clothing, features, expressions
4. Include the setting/mood for each scene

### Visual Style
- Cinematography reference (e.g., Vogue editorial, Terrence Malick)
- Color palette
- Lighting style
